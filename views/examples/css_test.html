CSS TEST

CSS defines some style in this way:
<style>
p {
color: red;
text-align: center;
}
</style>

i.e, css is to call any html tag followed with porperties using this syntax:
selector { property:value; } CSS selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute, and more.

Typically, you like to set some specific style not for a whole page and you like more than one style for some HTML element, ... so better use id-selector and ids:

this is red and centered

You can define a class with a '.' (dot) in front of a name and then use it in multiple types of HTML-elements by calling this calls

This is centered and red

comments within CSS blocks are defined between /* ... */
There are three ways to define a style:
  1. external, via a 'link' in the head section:
    <head> <link rel="stylesheet" type="text/css" href="mystyle.css"> </head>
  2. internal, via code in your header block:
    <head>
    <style>
      body {
         background-color: linen;
      }
    </style>
    </head>
  3. inline, i.e. within a HTML-element:
    <h1 style="color:blue;margin-left:30px;">This is a heading</h1>
    4. 'Predefined' colors are: Red, Green, Blue, Orange, Yellow, Cyan, Black
    so you can use them e.g. like:

    This text is red (color names are case insensitive)


    Alternatively, you can use RGB notation (style="color:rgb(0,200,55)"):

    This text is dark green (each color in rgb can be in [0,255])

    and RGB slider and the sentence below are found here: https://www.w3schools.com/css/css_colors.asp

    RGB values can also be specified using hexadecimal color values in the form: #RRGGBB, where RR (red), GG (green) and BB (blue) are hexadecimal values between 00 and FF (same as decimal 0-255).
