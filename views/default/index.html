Die Afropa Datenbank wird über diese Oberfläche administriert, bearbeitet und genutzt. Um die meisten Daten zu sehen, muss man eingeloggt sein. Einen Zugang (Account) bekommt man beim Administrator. Bitte immer daran denken, dass es sich um sensible, persönliche Daten handelt - d.h. bitte niemals z.B. ein Endgerät unbeobachtet in eingeloggtem Zustand zurücklassen.

Die Datenbank

Der Zugriff auf die Datenbanken erfolgt über {{=A('diesen Link', _href=URL(c='asyl', f='index'))}}. Es öffnet sich eine Index-Seite. Die Administrations- und Entwicklungswerkzeuge können ignoriert werden (und werden bald verschwinden). Wenn man die Datenbank öffnet, so erscheinen zunächst die Stammdaten aller Kunden. Neben diesen Stammdaten sind Links zu weiteren Daten, die dieser Person zugeordnet sind. Alle Daten können einzeln bearbeitet werden.

Einstellung der Sprache

Die Einstellung der verwendeten Sprache kann über das Hauptmenue oder über {{=A('diesen Link', _href=URL(f='my_language'))}} ausgewählt werden. Sollten noch Übersetzungen fehlen bzw. es Verbesserungsvorschläge gibt, bitte an den Administrator wenden.

Das Wiki

Die Basisversion eines Wikis ist ebenfalls implementiert und über das Hauptmenue oder {{=A('diesen Link',_href=URL(a='Asylum',c='default',f='wiki', args=['index']))}} erreichbar. Diese Funktionalität muss noch weiter ausgebaut werden, sollte mittelfristig ein Wiki gewünscht sein.
